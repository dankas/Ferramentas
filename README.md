# Ferramentas de suporte 
Ferramentas para suporte do parque de impressoras da compucom pelotas.

##Scripts de instalação no Linux e MACos
Os scripts de instalação executam o camando LPADMIN para instalar as impressoras pelo CUPS no Linux e MACos. Ambus precisam de permissão completa para usuario(CHMOD 700).
